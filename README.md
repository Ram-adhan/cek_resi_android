# Cek Resi Android

Aplikasi android untuk cek resi, menggunakan API dari binderbyte

Support pengecekan resi dengan kurir:
1. JNE - ok
2. SiCepat - ok
3. Ninja van - ok
4. Pos Indonesia - ok
5. Wahana - not tested
6. Tiki - not tested
7. anteraja - ok
8. JNT - ok

### Techstack:
1. Kotlin
2. MVVM-LiveData pattern
3. Rest-API with Retrofit
4. Dependency Injection with Dagger 2
