# Cek Resi Android

Aplikasi android untuk cek resi, menggunakan API dari binderbyte

Support pengecekan resi dengan kurir:
1. JNE - tested
2. SiCepat - tested
3. Ninja van - tested
4. Pos Indonesia - need fix
5. Wahana - not tested
6. Tiki - not tested
7. anteraja - need fix
8. JNT - need fix

### Techstack:
1. Kotlin
2. MVVM-LiveData pattern
3. Rest-API
